# Bug Hunting Lab

This workspace contains a small local bug-hunting lab for practicing debugging and verification.

## Run locally
- Start a server from this folder with: python3 -m http.server 8000
- Open: http://127.0.0.1:8000/index.html

## Files
- index.html: page layout and challenge UI
- app.js: form validation and bug fix target
- TASKS.md: task checklist
- BUG_REPORT.md: summary of the issue and fix
